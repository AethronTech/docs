# UBL-XML export

## Wat is UBL?

**UBL 2.1** (Universal Business Language) is het internationale standaardformaat voor elektronische facturen. Het is ook het formaat dat het Peppol-netwerk gebruikt.

## UBL-bestand downloaden

1. Ga naar **Facturen** en open de gewenste factuur
2. Klik op **Downloaden → UBL XML**
3. Je ontvangt een `.xml`-bestand

## Waarvoor gebruik je het UBL-bestand?

| Gebruik | Uitleg |
|---|---|
| Boekhoudpakket importeren | Exact, Winbooks, Yuki, ... kunnen UBL direct importeren |
| Sociaal secretariaat | Veel secretariaten accepteren UBL voor loonadministratie |
| Peppol Access Point | Verstuur de UBL via een access point naar de ontvanger |
| Archivering | Wettelijk geldige versie van je factuur |

## PDF downloaden

Naast UBL kun je ook een **leesbare PDF** downloaden:

1. Open de factuurdetailpagina
2. Klik op **Downloaden → PDF**

De PDF bevat alle wettelijk vereiste gegevens inclusief btw-berekening en factuurnummering.

## Factuurnummering

Klantel genereert automatisch opeenvolgende factuurnummers per jaar, bijv. **2026/0001**, **2026/0002**, ...

Het beginvolgnummer stel je in via **Instellingen → Facturatie → Startnummer**.

<Note>
Eenmaal een factuurnummer gebruikt, kan het niet worden hergebruikt. Dit is een wettelijke verplichting.
</Note>

# Projecten, taken en statussen

## Project aanmaken

1. Ga naar **Projecten** in de navigatie
2. Klik op **Nieuw project**
3. Vul een naam in en koppel het aan een **klant**
4. Optioneel: voeg een beschrijving en **uurtarief** toe
5. Klik op **Opslaan**

Het project krijgt automatisch de status **Te doen**.

## Projectstatussen

Elk project heeft één van deze drie statussen:

| Status | Betekenis |
|---|---|
| **Te doen** | Nog niet gestart |
| **Bezig** | Actief in uitvoering |
| **Voltooid** | Afgerond |

Wijzig de status via de projectdetailpagina → **Bewerken**.

## Taken beheren

Taken helpen je om werk binnen een project op te splitsen:

1. Open de projectdetailpagina
2. Scroll naar **Taken**
3. Klik op **Taak toevoegen**
4. Geef de taak een naam en optionele beschrijving

Bij het registreren van uren kun je uren aan een specifieke taak koppelen.

## Uurtarief instellen

Het uurtarief op een project wordt gebruikt bij het aanmaken van facturen op basis van geregistreerde uren:

1. Open de projectdetailpagina → **Bewerken**
2. Vul het veld **Uurtarief (€)** in
3. Klik op **Opslaan**

## Uren per project bekijken

Op de projectdetailpagina zie je:
- Totaal aantal geregistreerde uren
- Uitsplitsing per gebruiker
- Uitsplitsing per periode

<Tip>
De projectlijstpagina toont ook het totaal aantal uren per project — handig voor een snel overzicht.
</Tip>
